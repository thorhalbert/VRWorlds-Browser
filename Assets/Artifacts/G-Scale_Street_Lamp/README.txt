# Summary

This is a remix of a very, very nice lamp posted by SlashDev http://www.thingiverse.com/SlashDev/

My changes include:
- making the lamps a little taller
- adding an anchor helping in installing the lamps in a drilled hole
- adding a glass shade
- removal of some details (that would not print for me correctly no matter what)

More info here: http://my-miniature-worlds.blogspot.com/2017/02/my-first-3d-printed-g-scale-street-lamps.html

# Print Settings

Printer: Anet A6
Rafts: No
Supports: No
Resolution: 0.1mm
Infill: 10%

Notes: 
Print in a laying position.

Use transparent filament for the glass shade.
